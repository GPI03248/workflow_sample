# Release v1.0

CFWENO real-paper demo: from published PDF to validated scalar prototypes.

## Highlights

- **CFWENO3 scalar linear advection** — 3rd-order convergence (order 3.02), 380x more accurate than Lax-Wendroff
- **CFWENO3 scalar Burgers** — ~2nd-order convergence (order 2.01), 173x more accurate than Rusanov
- **Full paper-to-code workflow** exercised end-to-end across 13 gated stages
- **Three audit passes**: accuracy sweep, flux-form identity proof, SFM state-consistency verification

## Source Paper

- **Title**: High-order compact fully discrete schemes for inviscid compressible flows simulations
- **Authors**: Zhou, Dong, Pan
- **Journal**: Physics of Fluids 37, 106131 (2025)
- **DOI**: [10.1063/5.0291087](https://doi.org/10.1063/5.0291087)
- **Method**: CFWENO (Compact Fully-discrete WENO), 3rd-order stencil (Eq. 30)

## Validation Results

### Linear Advection (v1.1)

**Setup**: `u_t + a*u_x = 0`, `a = 1`, `u_0 = sin(2*pi*x) + 1`, periodic BC

| Method | L2 (nx=100) | Observed order |
|--------|------------|----------------|
| Upwind | 1.724e-02 | ~1 |
| Lax-Wendroff | 5.480e-04 | ~2 |
| **CFWENO3** | **1.444e-06** | **~3.02** |

- CFWENO3 is **380x** more accurate than Lax-Wendroff
- CFL sweep: stable at CFL 0.1, 0.5, 0.9

### Burgers Pre-Shock (v1.2)

**Setup**: `u_t + (u^2/2)_x = 0`, `u_0 = 1 + 0.2*sin(2*pi*x)`, CFL=0.5, T=0.15

| Method | L2 (nx=100) | Observed order |
|--------|------------|----------------|
| Rusanov baseline | 2.496e-03 | ~1.0 |
| **CFWENO3 Burgers** | **1.444e-05** | **~2.01** |

- CFWENO3 is **173x** more accurate than Rusanov at nx=100
- Predictor sweep (0/1/2 iterations): all yield ~2.0 order
- CFL sweep (0.1/0.3/0.5): all stable
- Reference sensitivity: order insensitive to reference grid (spread 0.04)

### Audit Findings

- **Flux-form audit (v1.2.1)**: `f_hat = f(ubar)` is an exact algebraic identity of the SFM two-step form for scalar Burgers — not an approximation
- **SFM state-consistency audit (v1.2.2)**: Spec defines `f*` at `ubar_{i+1/2}`, same state used in code — confirmed consistent

## Known Limitations

- **No full CFWENO** — scalar prototypes only (linear advection + Burgers pre-shock)
- **No Euler CFWENO** — characteristic decomposition not implemented
- **No 2D CFWENO** — dimensional extension not implemented
- **No CFWENO5/7** — 3rd-order stencil only
- **No post-shock validation** — requires nonlinear WENO weights (Eq. 17)
- **Burgers convergence is ~2nd order** — caused by per-cell `nu` variation in stencil derived for constant `nu`
- **Original PDF not redistributed** — publisher license restriction
